\textbf{微机结构}
\begin{itemize}
	\item Input 输入
	\item Output 输出
	\item Memory 存储器
	\item ALU 算术逻辑单元
	\item Control unit 控制单元
\end{itemize}

\textbf{指令集}：CISC(1-n个字)，RISC(1个字)

\textbf{字}：CPU一次可以处理的最大比特数

\textbf{位扩展}：同一地址的位扩展，满足一个字的输出

\textbf{字扩展}：增大字的量，选择不同的字，满足存储量需求
